This is a collection of proof-of-concepts, half-baked experiments, and "what if" files that fell out of a brain with too many browser tabs open. Part solution engineer, part chaos goblin, fully powered by curiosity and an unreasonable belief that everything can be automated.

If you're here, you'll find HTML files, scripts, AI experiments, home automation snippets, and things that technically work but probably shouldn't be shown in polite company. Every file here started as a passing thought that refused to stay passing.
